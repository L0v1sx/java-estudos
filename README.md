<h1>Calculadora de IMC em Java</h1>
<p>Projeto desenvolvido em Java para calcular o índice de Massa Corporal(IMC) com base no peso e a altura do usuário.</p>

<h1>Funcionalidades</h1>
<li><ol>Entrada de peso e altura </ol>
  <ol>Cálculo automático do IMC</ol>
  <ol>Classificação do Resultado (Abaixo do peso,sobrepeso etc..)</ol>
  <ol>Repetição do Calculo (Ainda em melhorias)</ol>
</li>

<h1>Fórmula utilizada</h1>
<p>IMC = peso / (altura * altura)</p>

<h1>Tecnologias</h1>
<li><ol>Java</ol>
  <ol>Programação orientada em Lógica básica</ol>
  <ol>Uso de estrutura de Repetição (Ainda no processo)</ol>
</li>
